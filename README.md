# 5-bit-ALU-VHDL

This is a VHDL ALU Project that is a part of Digital Design Course that performs operations like:
### Arithmetic Operations
- sel     Operation          
- 0000    increment a
- 0001    decrement a
- 0010    transfer b
- 0011    increment b      
- 0100    decrement b
- 0101    transfer a
- 0110    add a and b
- 0111    multiply a by 2


### Logical Operations
- 1000    complement a (1s complement)
- 1001    complement b
- 1010    AND
- 1011    OR
- 1100    XOR                      
- 1101    XNOR
- 1110    NAND
- 1111    NOR
